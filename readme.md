# ML-based Automatic Emergency Braking (AEB)

This project trains machine learning models for Automatic Emergency Braking using the MetaDrive simulator.

- Collects simulation data (ego speed, relative speed, distance, TTC).
- Trains classification (brake / no-brake) and regression (brake value) models.
- Evaluates with metrics and visualizations inside the notebook.
- Includes groundwork for closed-loop AEB behavior in simulation.
